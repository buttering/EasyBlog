# 什么是风险
可能给项目的成功带来威胁或损害的可能性。
# 风险管理的目标
对项目目标而言的正面事件的概率和影响结果最大化，把负面事件的概率和影响最小化。
# 风险承受度
从潜在回报中得到满足或快乐的程度。
# 应对风险的基本措施
- 规避
- 接收
- 减轻
- 转移
# 什么是项目采购
通过努力从项目组织外部获得物料、工程和服务的 整个采办过程。
